# Nexcent Landing Page

A minimal agency website landing page built with pure HTML and CSS.

## Live Preview

Open `index.html` in your browser.

## Tech Stack

- HTML5 (semantic)
- CSS3 (custom properties, flexbox, grid)
- [Swiper.js](https://swiperjs.com/) v11 — hero slider
- [Inter](https://fonts.google.com/specimen/Inter) — Google Fonts
